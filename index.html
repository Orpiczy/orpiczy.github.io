CSS Display and Position CheatSheet

Simple cheatsheet for css's display and position attributes

Display

DISPLAY
TYPE DESCRIPTION
block element box taking full width and height of its container, generates line breaks
inline element box taking as little space as possible, no line breaks and height and width adjustment
inline-block element box with height and width adjustment and with no line breaks
cheatsheet no.1

Position

POSITION
TYPE DESCRIPTION
static element is position according to normal document flow -> top, right, bottom, left arguments doesn't matter, it's a default position value
relative element is position according to normal document flow, but top, right, bottom, left offsets does matter, element's position does not impact position of other elements
absolute element is removed from normal document flow and positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block, it's final position is determined by top, right, bottom, left offset values
fixed element is removed from normal document flow and positioned relative to initial containing block, element is removed from normal document flow, it's final position is determined by top, right, bottom, left offset values
sticky element is part of normal document flow and position relative to closest scrolling ancestor and containing block
cheatsheet no.2
